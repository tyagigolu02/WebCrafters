Spotify Clone

Developed a Spotify clone web application that mimics the core functionalities and user interface of the popular music streaming service. The project aimed to provide users with a seamless music streaming experience, including features like browsing playlists, searching for songs, and playing music.

Technologies Used

  • Frontend: HTML, CSS, JavaScript, React
  • Backend: Node.js, Express
  • Database: MongoDB
  • State Management: Redux
  • Authentication: Firebase Authentication
  • Deployment: Heroku
  • Music Streaming API: Spotify Web API
