Weather App

Developed a weather app that provides real-time weather information and forecasts for any location. The app allows users to search for a city and view current weather conditions, including temperature, humidity, wind speed, and weather descriptions.

Technologies Used

  • Frontend: HTML, CSS, JavaScript, React
  • Backend: Node.js, Express
  • Database: MongoDB
  • State Management: Redux
  • API: OpenWeatherMap API
  • Deployment: Heroku
