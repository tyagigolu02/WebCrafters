Netflix Clone

Developed a Netflix clone web application that replicates the user interface and core functionalities of the popular streaming service. The project aimed to provide a seamless streaming experience with a user-friendly interface, allowing users to browse, search, and watch videos.

Technologies Used

  • Frontend: HTML, CSS, JavaScript, React
  • Backend: Node.js, Express
  • Database: MongoDB
  • State Management: Redux
  • API: TMDB (The Movie Database) API
  • Deployment: Heroku
