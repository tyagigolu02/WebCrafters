Hotstar Clone

Developed a Hotstar clone web application that replicates the user interface and core functionalities of the popular streaming platform, Hotstar. The project aimed to provide users with a seamless streaming experience, including features like browsing content, searching for shows and movies, and streaming videos.

Technologies Used

  • Frontend: HTML, CSS, JavaScript, React
  • Backend: Node.js, Express
  • Database: MongoDB
  • State Management: Redux
  • API: TMDB (The Movie Database) API
  • Authentication: Firebase Authentication
  • Deployment: Heroku
